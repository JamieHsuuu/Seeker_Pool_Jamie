# Seeker_Pool_Jamie
## 專題介紹
以**MVC架構**為基礎，使用**Java Servlet**, **JDBC**等技術且**前後端分離**的人力銀行求職網站。
## 使用技術
- 前端：HTML, CSS, JavaScript, JQuery, Bootstrap, Axios  
- 後端：Java Servlet, Java Model  
- 資料庫：MySQL, Redis（使用於註冊帳號時須填寫的驗證碼）  
## 功能說明
**【企業前台】**
1. 企業會員註冊
2. 企業會員登入、登出
3. 企業會員資料管理
4. 應徵者管理
5. 取消面試
6. 完成面試

**【一般前台】**
- 瀏覽企業資訊  

**【後台】**  
- 企業會員管理
